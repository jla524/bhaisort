# bhaisort

[Bubble sort][1] in [Bhailang][2].

## Quickstart Guide

```sh
git clone https://github.com/aditya87/bhai-lang.git
cd bhai-lang
git checkout aadi-arrays
npm install
npm run build
cd packages/cli
npm link
cd ../../..
bhailang bubblesort.bhai
```

[1]: https://www.youtube.com/watch?v=n8kWyYstCh8
[2]: https://bhailang.js.org/#playground
